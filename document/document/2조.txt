숙소 예약 프로그램

개요
부킹닷컴 트리바고 스카이스캐너 등의 기존의 숙소예약 및 항공권 예약 사이트를 참고하여 숙소예약 사이트를 만들어보고자 한다.


주요 기능

1. 회원 및 정보 관리
 - 회원은 사용자, 관리자 두 경우로 분류
 - 숙소의 정보 등록 및 관리

1). 사용자
 - 사용자는 회원가입을 하여 웹 페이지의 기능을 이용할 수 있음
 - 사용자는 숙소를 이용하는 이용자가 될 수 있음
 - 사용자는 숙소를 제공하는 사업자가 될 수 있음

2). 이용자
 - 검색, 숙소 예약 등의 전반적인 기능을 이용

3). 사업자
 - 숙소 등록하기 버튼을 통해 숙소등록 페이지로 이동하여 숙소 등록 가능

4). 숙소
 - 숙소 유형
 - 숙소 기본 정보 ( 소유자, 주소 등)
 - 주소 제공 시 구글맵 API를 통해 구현
 - 구성 및 요금
 - 시설 및 서비스
 - 편의시설
 - 사진
 - 결제





2. 메인 페이지

1). 검색
 - 검색 키워드에 따른 도시, 국가를 한꺼번에 검색
 - 날짜 선택은 캘린더 형태로 제작
 - 인원 수 선택에 따른 결과

2). 숙소 유형별 예약
 - 호텔, 리조트, 아파트, 빌라, 샬레, 코티지, 글램핑, 아파트호텔, 펜션, 게스트하우스, 호스텔, 모텔, 베드앤브렉퍼스트, 료칸, 리야드, 홀리데이파크, 홈스테이
   캠프장, 컨트리하우스, 팜스테이, 보트호텔, 럭셔리텐트, 취사가능숙소, 타이니하우스

3). 숙소 지역별 예약
 - 이용자의 접속 지역에 따라 그 지역에서 가장 많이가는 지역별로 정렬하여 보여줌

4). 추천 여행지
 - 지역, 도시, 관광명소 별 제공
 - 추천여행지를 보여주기위한 알고리즘은 고려할 필요가 있음

5). 구독, 초대메일 등을 위한 메일링 시스템
 - 메일을 입력 시 실제 메일로 발송이 되는 기능
 - 할인쿠폰등을 제공하기 위한 기능





3. 검색 결과페이지

1). 검색 결과 내 세부 검색

2). 검색된 결과 소팅
 - 요금 순
 - 성급 순
 - 성급 + 요금
 - 도심으로부터 거리
 - 평점 순
 - 후기 수 순

3). 검색 결과 필터링
 - 성급, 도심으로부터 거리, 주변시설, 접근성, 유형, 평점, 시설, 객실, 편의시설 등의 항목






4. 숙소
 - 사진, 숙소설명, 시설 등의 숙소 정보를 표현
 - 예약 기능
 - 결제
 - 후기




추가사항

- 최대한 부킹닷컴 등의 기존의 페이지의 기능을 모두 포함하도록 구현
- 사업자의 기존 데이터베이스와 연동기능을 구현하기 위해 테이블을 공유하여 사용
- 기존의 사업자의 사이트를 보여주기용으로 제작
- UML 단계에서부터 데이터베이스의 데이터를 만들어라.
